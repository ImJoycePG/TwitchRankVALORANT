English

How to Use the Twitch Rank VALORANT Link

This link allows you to retrieve information about a VALORANT player's rank on Twitch.

Parameters:

  • api_key: Your API key obtained from Henrik Dev's VALORANT API.
  • name: The player's name.
  • tag: The player's tag.
  • region: The player's region (e.g., latam, na, eu).

Usage:

Replace the placeholders (YOUR_API_KEY, PLAYER_NAME, PLAYER_TAG, REGION) in the link with your actual API key and the player's information.

Example:

https://twitchrankvalorant.glitch.me/rank?region=REGION&name=PLAYER_NAME&tag=PLAYER_TAG&api_key=YOUR_API_KEY

Replace YOUR_API_KEY with your actual API key, PLAYER_NAME with the player's name, PLAYER_TAG with the player's tag, and REGION with the player's region.

Objective

This was specifically created so that content creators can use it in their livestreams, as some chatbots like Nightbot currently have a character limit.

Reemplaza los marcadores de posición (YOUR_API_KEY, PLAYER_NAME, PLAYER_TAG, REGION) en el enlace con tu clave de API real y la información del jugador.

https://twitchrankvalorant.glitch.me/rank?region=REGION&name=PLAYER_NAME&tag=PLAYER_TAG&api_key=YOUR_API_KEY

Reemplaza YOUR_API_KEY con tu clave de API real, PLAYER_NAME con el nombre del jugador, PLAYER_TAG con la etiqueta del jugador y REGION con la región del jugador.
